# FrescoGO!

```
CTRL-0                  edita o nome do árbitro
CTRL-1                  edita o nome do atleta à esquerda
CTRL-2                  edita o nome do atleta à direita
CTRL-I                  inverte a posição dos atletas

CTRL-SPACE              inicia uma sequência
CTRL-ALT-SPACE          marca uma queda de bola
CTRL-ALT-BACKSPACE      volta atrás e descarta a última sequência
CTRL-R                  reinicia a apresentação

<-                      marca golpe do atleta à esquerda
  ->                    marca golpe do atleta à direita
```
