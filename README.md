# FrescoGO!

```
CTRL_0              edita o nome do árbitro
CTRL_1              edita o nome do atleta à esquerda
CTRL_2              edita o nome do atleta à direita
CTRL_I              inverte a posição dos atletas

CTRL_↑              inicia uma sequência
CTRL_↓              marca uma queda de bola
← | →               marca um golpe do atleta à esquerda ou à direita

CTRL_- | CTRL_+     remove ou adiciona uma queda
CTRL_BACKSPACE      volta atrás e descarta inteiramente a última sequência
CTRL_R              reinicia a apresentação
CTRL_ESCAPE         fecha o programa
```
